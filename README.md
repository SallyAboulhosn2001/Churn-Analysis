# Churn-Analysis
This project focuses on analyzing customer churn using a real-world telecom dataset and applying machine learning techniques to identify the key factors that lead customers to leave.

## What was done
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA) to understand churn patterns
- Feature engineering
- Training and evaluating churn prediction models

## Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Project Outputs
- Jupyter notebook containing the full analysis and modeling
- PDF report summarizing the main insights and results

## Goal
The goal of this project is to demonstrate an end-to-end data analysis and machine learning workflow, from raw data to actionable insights.
